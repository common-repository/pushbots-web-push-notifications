Reach out to your Wordpress visitors with desktop push notifications. Now supporting Chrome, Firefox, Opera and Safari.

== Description ==
Increase user engagement, retention and life-time-value for your app and website via the power of push messaging. This plugin uses PushBots.com service and requires you to have a working account.  With the plugin and a PushBots account, you will be able to reach out to your Wordpress visitors with desktop push notifications. Now supporting Chrome, Firefox, Opera and Safari.

== Installation ==

1. Install PushBots from the WordPress.org plugin directory or by uploading the PushBots plugin folder to your wp-content/plugins directory.
2. Active the PushBots plugin from your WordPress settings dashboard.
3. Follow the instructions on the Setup page.
